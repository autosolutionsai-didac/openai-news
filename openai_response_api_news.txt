OpenAI Response API News (March 20, 2025)

1. OpenAI has introduced the Responses API, which appears to be a new offering designed to replace the current Assistants API.

2. According to the search results, the Responses API was announced on March 11, 2025, and is described as "a faster, more flexible, and easier way to create agentic experiences that combines the simplicity of Chat Completions."

3. The Responses API includes built-in tools like web and file search capabilities, as well as the ability to perform tasks on a computer.

4. The Responses API is available to all developers at no additional cost.

5. OpenAI plans to retire the current Assistants API in the first half of 2026, with the Responses API eventually replacing it.

6. Along with the Responses API, OpenAI has also unveiled an open-source Agents SDK to help developers build their own AI agents more efficiently.
